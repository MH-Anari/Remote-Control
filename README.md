# Remote-Control
Control your computer remotely

## Install server
```
git clone https://github.com/MH-Anari/remote-control.git
cd remote-control/server
pip install -r requirments
python server.py
```

## Install client
```
git clone https://github.com/MH-Anari/remote-control.git
cd remote-control/client
python client.py
```
